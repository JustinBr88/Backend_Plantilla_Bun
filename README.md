# Backend Plantilla Bun

Plantilla base para iniciar un backend con Bun y TypeScript.

## Requisitos

- Bun instalado (recomendado version 1.3.12 o superior)

Verificar instalacion:

```bash
bun --version
```

## Instalacion

Instala las dependencias del proyecto:

```bash
bun install
```

## Ejecucion

Inicia el proyecto en modo simple:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.12. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
